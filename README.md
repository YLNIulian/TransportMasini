# TransportMasini
Să se realizeze o aplicație pentru o companie de transporturi. Se va considera evidența conducătorilor auto, a mașinilor disponibile și a intervalelor de lucru pentru fiecare. # TransportMasini - Sistem de Gestionare Transport

 Descriere Proiect

Aplicație pentru o companie de transporturi care permite gestionarea eficientă a:
- Conducători auto - evidența personalului
- Mașini disponibile - inventarul vehiculelor
- Intervale de lucru - programul și disponibilitatea
- Kilometri parcurși - urmărirea consumului și usturii
- Trasee - istoricul deplasărilor

 Caracteristici Principale
 1. Gestionarea Conducătorilor Auto
- Adăugare/ștergere conducători
- Vizualizare informații personale
- Urmărire experiență și certificări
- Istoric deplasări pe conducător

2. Gestionarea Vehiculelor
- Evidența mașinilor disponibile
- Urmărire stare și condiție
- Planificarea întreținerii
- Calculul kilometrilor totali

3. Intervale de Lucru
- Programare ture pentru conducători
- Asignarea vehiculelor la ture
- Urmărire ore lucrate
- Planificarea disponibilității

4. Urmărire Kilometri
- Înregistrare km parcurși pe mașină
- Calculul km pe conducător
- Estimare consum combustibil
- Rapoarte de utilizare

5. Istoric Trasee
- Vizualizare rute completate
- Detalii despre fiecare deplasare
- Puncte de plecare și sosire
- Durata și consum resurse
